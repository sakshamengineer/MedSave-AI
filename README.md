## 🏆 Hackathon Project

# MedSave AI 💊

MedSave AI is an AI-powered healthcare application that helps users **save money on medicines** by recommending affordable **Jan Aushadhi** alternatives for medicines prescribed by doctors.

Users simply upload a prescription, and MedSave AI analyzes it to identify medicines, compare them with equivalent Jan Aushadhi medicines, calculate potential savings, and help users locate the nearest Jan Aushadhi stores.

---

## 🚀 Features

### 📄 Prescription Analysis

* Upload a doctor's prescription.
* AI extracts the prescribed medicines from the prescription.

### 💊 Jan Aushadhi Alternatives

* Finds equivalent **Jan Aushadhi** medicines for each prescribed medicine.
* Displays both branded and Jan Aushadhi medicine names.

### 💰 Price Comparison

* Compares the price of branded medicines with Jan Aushadhi alternatives.
* Calculates:

  * Total cost of branded medicines
  * Total cost of Jan Aushadhi medicines
  * Amount saved
  * Percentage savings

### 📍 Nearby Jan Aushadhi Store Locator

Users can locate Jan Aushadhi Kendras in two ways:

* **Using Location Access:** Automatically finds nearby Jan Aushadhi stores based on the user's current location.
* **Manual City Search:** If location permission is denied, users can simply enter a city name to search for nearby Jan Aushadhi stores.

### 🤖 AI-Powered Assistance

* Intelligent prescription analysis
* Medicine identification using AI
* Fast and user-friendly experience

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
cd MedSave-AI
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file and configure the required environment variables.

Run the application:

```bash
python main.py
```

## 🎯 Mission

Millions of people spend more on medicines simply because they are unaware of affordable alternatives. MedSave AI aims to bridge that gap by helping users discover equivalent Jan Aushadhi medicines, compare prices, calculate their savings, and easily locate nearby Jan Aushadhi stores, making healthcare more affordable and accessible.
